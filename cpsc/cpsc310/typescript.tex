\section{Typescript basics}
No point in duplicating a lot of notes between this doc and the Typescript handbook so I'll just record some small notes. \\
~\\
Little translations between Typescript and java
\begin{itemize}
	\item Type annotations like rust
	\item "export default" instead of "public"
	\item constructor() function for the constructor definition
	\item Logger: Log.trace for our project, console.log usually
	\item For each loops: for(thing of list){}
	\item == for truthiness, === for strict equality
\end{itemize}
Declaring variables
\begin{itemize}
	\item const: value is mutable, but can't be reassigned
	\item let: reassignable
	\item var: never use this (puts it in a global scope?)
\end{itemize}
